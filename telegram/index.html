Telegram 新手完整使用教程

首先介绍一下 Telegram，可能很多人都没听说过，这款软件简洁易用，没有广告，没有朋友圈红包钱包理财的功能，只有聊天，只有聊天，只有聊天。
客户端基本上全平台适配，包括某 1% 平台。（Symbian 用户就不要来凑热闹了）

1.使用前必要的准备

由于我们不知道的原因，Telegram 无法在中国一些地区正常注册、登录，所以我们需要下载一个代理软件。
(如果你已经有代理软件或者你可以自由访问互联网可以不用下载下列软件)
推荐：Lantern Windows OS X Android
Psiphon 赛风 Windows
Android
iOS 用户可以尝试 App Store 中各种免费 VPN。

建议全部下载，以免其中一个软件因不可抗力无法继续使用。

如果你使用 Psiphon，建议在设置中开启「拆分隧道」（国内外自动分流），点击「更改」保存设置。

下载与注册 Telegram

Windows
OS X 请在 Mac App Store 搜索「Telegram Desktop」
iOS 在 App Store 中搜索「Telegram」
Android 用户可以在「酷市场」酷安中下载。

一步步 Next 即可。
注册时如果收不到验证码，请将代理软件临时切换至「全局模式」重试

汉化 Telegram

打开 Telegram zh-cn Project 的公众号（或者称频道 Chanel），加入（JOIN）频道，按照提示下载配置文件，并按照提示安装即可。（Desktop 用户请加载文件后不要删除文件）

开始聊天

你可以在设置页面设置一个用户名，别人用用户名就可以与你对话，你也可以将你的聊天网址发给他人： http://telegram.me/用户名

由于 Telegram 有大量国外（尤其是中东地区）用户，你可能会收到陌生人各种语言的骚扰，你可以点开头像，「屏蔽」(block)该用户，但请不要滥用屏蔽功能，屏蔽一个用户可能会导致他的帐号发消息受限。

创建群组


点击右边的按钮即可创建群组和频道
频道是类似微信订阅号的东西，但是你只能查看频道的内容，而不能向频道发送消息，

建立新的群组后，你可以通过群资料页面的邀请链接邀请其他 Telegram 用户进群，如果该链接被滥用，你也可以更换一个链接，原链接将失效。

一般的群只能容纳 200 人，但是群主可以在群简介中输入「tosupergroup」来升级为超级群，超级群的消息可由作者和管理员撤回（普通群管理员可以删除消息），超级群可以看到之前的聊天记录，请谨慎升级。升级会创建新的加群链接，原链接失效

加好友

在 Telegram 中，你知道对方的用户名就可以开始对话，但你如果想把他添加到你的通讯录中，就必须得到他的手机号，使用「联系人」中的「添加联系人」即可。将他人添加进你的通讯录，他会看到你的手机号，请注意隐私保护

私密对话

点击他人头像点击「开始私密对话」且对方在线即可进入端到端加密的安全对话， 可以设置阅后即焚，Telegram 不保存任何聊天记录，传输全程加密。

OS X 的 Telegram Desktop 并不能使用此功能，你可以使用 Mac App Store 里的「Telegram」客户端（没有Desktop）。

文件传输

Telegram 支持最高 1.5 G 的任意格式文件传输，请不要滥用，接受他人文件前也请格外小心。
